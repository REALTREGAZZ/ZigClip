# Video Assets

Place your MP4 video clips in this directory with the following names:

- `clip_1.mp4`
- `clip_2.mp4`
- `clip_3.mp4`
- `clip_4.mp4`
- `clip_5.mp4`

## Requirements

- Format: MP4 (H.264)
- Duration: 12-20 seconds recommended
- Resolution: 720p or 1080p
- Aspect Ratio: 16:9 or 9:16 (portrait)

## For Testing

If you don't have actual gaming clips yet, you can use any short video files for testing purposes.
Just ensure they are named correctly as listed above.
