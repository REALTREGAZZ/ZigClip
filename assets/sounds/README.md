# Audio Assets

Place your audio file in this directory:

- `kill_sound.mp3`

## Requirements

- Format: MP3
- Duration: ~200ms (short sound effect)
- Type: Victory/kill sound effect

## For Testing

If you don't have a kill sound yet, you can use any short audio clip or find free sound effects from:
- freesound.org
- zapsplat.com
- mixkit.co

Just ensure the file is named `kill_sound.mp3`.
